# Nakagaki Yuto Portfolio

自身のポートフォリオになるようなサイトを作りました。<br><br>
【URL】[Nakagaki Yuto Portfolio](https://nakagakiyuto.pythonanywhere.com/)」<br>
 
# Features

* トップページ<br>
　サイトの紹介、成果物、現在取り組んでいること、持っている・勉強中のスキル、ちょっとしたブログ
* ブログ投稿機能（管理者のみ）

* コメント投稿機能（全ユーザ）

 
# Requirement
 
* Django 2.2.4
 
# Installation

```bash
pip install Django~=2.2.4
```
 
# Usage
 
```bash
git clone https://github.com/Nakagaki-Yuto/Portfolio.git
cd Portfolio
python manage.py runserver
```
